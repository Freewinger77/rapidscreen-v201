# Recruitment Platform Plan

## User Request
Create a recruitment platform with a job details page featuring:
- Kanban board with swimlanes for candidate management
- Drag-and-drop functionality to move candidates between swimlanes
- Ability to create new candidate tickets
- Light and dark mode support
- Design based on provided reference image
